# Asya Pc Bot 🖥️🎵

Современный и безопасный Telegram-бот на базе `aiogram 3.x` для удаленного управления вашим Windows-компьютером с помощью интерактивного Inline-меню.

---

## 🌟 Основные возможности

* **📺 Интерактивное Inline-меню**: Удобное переключение категорий и действий внутри одного сообщения без засорения истории чата.
* **🚀 Динамический запуск программ**: Настройка списка запускаемых приложений и путей к ним в отдельном файле конфигурации `programs.json` (изменения подгружаются на лету без перезапуска бота). Поддерживаются переменные окружения Windows (например, `%APPDATA%`).
* **❌ Закрытие программ и процессов**: Возможность принудительно завершить процесс программы (через `taskkill` с автоматическим обходом ограничений через `tskill` для Диспетчера задач) или закрыть текущее активное окно (Alt+F4) в 1 клик.
* **🎵 Управление мультимедиа**: Глобальный пульт управления воспроизведением на ПК (плей/пауза, переключение треков, регулировка громкости шагом по 2%, Mute). Работает во всех приложениях (Spotify, Яндекс.Музыка, YouTube в браузере, VLC и др.).
* **📸 Скриншоты экрана**: Мгновенное создание снимка экрана и его отправка в чат (с использованием оптимизированного сохранения в буфер оперативной памяти).
* **⚙️ Системные действия**:
  * Выключение ПК
  * Спящий режим
  * Блокировка экрана (Win + L)
  * Быстрый запуск Диспетчера задач
* **🔒 Повышенная безопасность (Middleware)**: Бот авторизует запросы только от разрешенных Telegram User ID (настраивается в `.env`). Неавторизованные пользователи увидят предупреждение, а их попытки нажать на Inline-кнопки будут заблокированы.
* **🛡️ Защита от сбоев и спама**:
  * **Single Instance Lock**: Блокировка запуска дубликатов процесса бота с помощью файлов блокировок (`msvcrt`).
  * **Регулировка частоты (Cooldown)**: Ограничение автоматической отправки уведомлений о включении ПК (не чаще 1 раза в 5 минут).
  * **Самовосстановление сети**: При включении компьютера бот будет автоматически пытаться подключиться к сети в течение 3 минут (с шагом 10 секунд), что решает проблему долгой инициализации интернета на старте Windows.

## 🛠️ Установка и запуск

### 1. Требования
* Операционная система: **Windows**
* Установленный **Python 3.8+**

### 2. Клонирование репозитория и установка зависимостей
Создайте папку проекта, инициализируйте виртуальное окружение и установите библиотеки:
```bash
git clone https://github.com/yourusername/telegram-pc-bot.git
cd telegram-pc-bot

# Создание виртуального окружения
python -m venv venv

# Активация виртуального окружения
venv\Scripts\activate

# Установка зависимостей
pip install -r requirements.txt
```

### 3. Конфигурация бота
1. Создайте файл `.env` на основе шаблона `.env.example`:
   ```bash
   copy .env.example .env
   ```
2. Откройте `.env` и введите:
   * `BOT_TOKEN` — ваш токен бота от [@BotFather](https://t.me/BotFather).
   * `ALLOWED_USERS` — ваш цифровой Telegram User ID (или несколько через запятую).
3. Настройте список программ для быстрого запуска в файле `programs.json`:
   ```json
   [
     {
       "name": "Блокнот",
       "path": "notepad.exe",
       "process": "notepad.exe"
     },
     {
       "name": "Google Chrome",
       "path": "C:\\Program Files\\Google\\Chrome\\Application\\chrome.exe",
       "process": "chrome.exe"
     },
     {
       "name": "Telegram",
       "path": "%APPDATA%\\Telegram Desktop\\Telegram.exe",
       "process": "Telegram.exe"
     }
   ]
   ```

### 4. Запуск
Для запуска бота просто запустите файл `start.bat` двойным кликом или выполните команду в консоли:
```bash
python bot.py
```
